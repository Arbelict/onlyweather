# SkyPulse Weather 实时天气网页

这是一个基于用户实时位置的天气网页应用，显示当前及未来一周的天气信息。

## 功能特色

- 🌍 **实时定位**：自动获取用户位置并显示当地天气
- 📊 **丰富的界面**：包含温度、湿度、风速、气压、空气质量等多项指标
- 📈 **图表可视化**：温度趋势图表
- 🌅 **日出日落**：可视化时间线
- 🗓️ **一周预报**：未来7天天气预测
- 🌿 **空气质量**：PM2.5、PM10、臭氧、NO₂等空气指标
- 📱 **响应式设计**：适配手机和桌面设备

## 使用方法

### 1. 直接打开HTML文件
直接将 `index.html` 文件拖拽到浏览器窗口中即可使用。

### 2. 本地运行服务器
```bash
# 安装依赖
npm install

# 启动服务器
npm start
```
然后在浏览器打开 http://localhost:3000

### 3. 在线托管（推荐）
将整个文件夹上传到：
- GitHub Pages
- Netlify
- Vercel
- 或其他静态网页托管服务

## API使用
网页使用 [Open-Meteo](https://open-meteo.com/) API 获取天气和空气质量数据，无需API密钥。

## 界面预览

应用采用深色主题设计，包含以下模块：
- **主卡片**：当前温度、天气状态、体感温度
- **空气质量卡片**：AQI指数和污染物浓度
- **24小时预报**：每小时温度变化
- **未来一周预报**：每天最高/最低温度
- **温度趋势图表**：24小时温度曲线
- **气象指标**：风速、湿度、气压等
- **日出日落时间线**：可视化一天时间进度

## 技术细节
- 纯HTML/CSS/JavaScript实现
- 使用Chart.js绘制温度图表
- 使用Open-Meteo免费API
- 响应式设计适配移动设备
- 无需后端服务器（可直接静态部署）

## 部署指南

### Netlify部署（最简单）
1. 访问 https://netlify.com
2. 注册账号
3. 点击"New site from Git"
4. 选择weather-app文件夹
5. 点击部署

### GitHub Pages部署
1. 创建GitHub仓库
2. 上传weather-app文件夹
3. 设置Settings → Pages → Source: main branch
4. 等待部署完成
